A Wordpress plugin that add Instagram Embed post using Gutenberg editor.


=== Description ===

Instagram Post Embed is a powerful WordPress plugin that allows you to seamlessly embed Instagram posts directly into your website or blog. With its easy-to-use interface, you can display Instagram posts with captions, comments, and likes, making your website more engaging and interactive. The plugin offers various customization options, such as choosing the size and layout of the embedded post, and it's fully responsive, ensuring that it looks great on all devices. Whether you're a blogger, influencer, or business owner, Instagram Post Embed is the perfect tool to enhance your website with stunning Instagram content.

=== Installation and Use ===

1. Log in to your WordPress admin dashboard.
2. Go to "Plugins" > "Add New".
3. Search for "Instagram Post Embed" in the plugin search bar.
4. Click the "Install Now" button next to the plugin.
5. Once the plugin is installed, click the "Activate" button to activate it.
6. After activating the "Instagram Post Embed" plugin, go to the post or page where you want to embed an Instagram post.
7. In the WordPress editor, switch to the Gutenberg editer view.
9. Copy the Instagram post URL that you want to embed from Instagram.
10. Paste the Instagram post URL into the INSTAGRAM URL input field, in the desired location where you want the post to appear.
11. Save or update the post/page to see the embedded Instagram post.

=== It's not working ===

- Step 1. Check if your plugin is activated.
- Step 2. Deactivate all plugins and reactivate Block collection.
